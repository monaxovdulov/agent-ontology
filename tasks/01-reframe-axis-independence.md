# 01 - Reframe Axis Independence

## Priority

Wave 1: fix false claims.

## Problem

The document claims there are seven orthogonal axes, but that is not accurate.

- A7 "properties" is marked as derivative, so it should not be treated as a core axis.
- A1 "composition" overlaps with A6 "technical layer".
- A5 "roles" partly derives from composition.
- A2, A3, and A4 appear to be the strongest independent axes.

## Task

Replace "orthogonal axes" with a more precise claim such as "partially independent dimensions".

## Acceptance Criteria

- The text no longer claims that all seven axes are orthogonal.
- A2 x A3 x A4 are identified as the most independent dimensions.
- A1, A5, A6, and A7 are described as overlapping or derived dimensions.
- Any visual labels are updated to match the revised claim.

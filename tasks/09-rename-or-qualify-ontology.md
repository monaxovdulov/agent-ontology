# 09 - Rename Or Qualify Ontology

## Priority

Wave 3: make the framing honest.

## Status

Done.

## Problem

The artifact calls itself an ontology, but it is currently closer to a taxonomy. A real ontology would define formal relations, axioms, and inference rules.

## Task

Either rename the artifact to "taxonomy" or explicitly state the limitations of using "ontology".

## Acceptance Criteria

- [x] The title and introduction no longer oversell the artifact.
- [x] If "ontology" remains, the document explains that it is an informal ontology/taxonomy.
- [x] The document distinguishes taxonomy from formal ontology.
- [x] Strong ontology language in title, hero, section headings, and summary is removed or explicitly qualified.

## Notes

- Renamed the visible framing from `чистая онтология` to `практическая таксономия` in the page title and hero.
- Reworked the lead and summary to state that the artifact is a taxonomy / map of dimensions with consistency checks, not a formal ontology with axioms and inference rules.
- Replaced `Онтологический дребезг` and `ось онтологии` with category/map language.
